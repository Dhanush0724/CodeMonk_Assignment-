✅ Backend API is Running

You can use Postman to test the following endpoints:

👉 Download the Postman collection file: CodeMonk_Assignment.postman_collection.json
👉 Open Postman and click Import
👉 Choose the downloaded collection file
👉 You'll see all API endpoints organized under CodeMonk_Assignment
💡 Make sure to set the JWT token in the "Authorization : Bearer your_access_token" header after login for protected routes of paragraphs and search.
⚠️ If your token has expired or returns a 401 error:
- Login again using the /api/login/ endpoint
- Copy the new access token from the response
- In Postman, go to the headers of protected routes (/api/paragraphs/ and /api/search/)
- Add this header: Authorization: Bearer <your_access_token>

  • POST /api/register/ – Register a user
  • POST /api/login/ – Login to get JWT token
  • POST /api/paragraphs/ – Submit paragraphs
  • GET /api/search/?word=hello – Search paragraphs

✅ You're now ready to test the API securely.
